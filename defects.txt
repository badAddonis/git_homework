Файл cmd.html.
1. Лишний пробел в подвале страницы. Сейчас "202 4", а должно быть "2024".
2. DIR - Изменить каталог - неверно. DIR показывает содержимое папки, а не меняет каталог. Для смены каталога используется CD.
3. CD - Отображение файлов и папок в текущем каталоге - неверно. CD меняет текущий каталог.
4. MAKEDIR - Создать папку - неверно. В Windows CMD обычно используется MD или MKDIR, а не MAKEDIR.
5. DATE - Выводит или устанавливает системное время - неверно. DATE работает с датой.
6. TIME - Выводит или устанавливает текущую дату - неверно. TIME работает со временем.
7. Добавить кнопку для возвращения на главную страницу.

Файл mac.html.
1. ls - это не "List files in the directory". Это сокращение от list, то есть "список". Команда показывает список файлов в папке.
2. pwd - обычно расшифровывается как Print working directory, а не Present working directory.
3. mv - это не Make directory, а move. Перевод "создать папку" неверный.
4. mkdir - это не Move, а make directory или make directories. Перевод "переместить" неверный.
